Problem setup

Given a directed graph $G=(V, E)$ with edge capacities $c_e$ for each edge $e$, construct a $s-t$ flow from a source vertex $s$ and a sink vertex $t$.

What is a flow?

A $s-t$ flow is defined as a function $f: E \rightarrow R^+$ that satisfies 2 conditions:

1. Capacity: $0 \leq f(e) \leq c_e$ for each edge $e$. Meaning the flow on each edge should not exceed its capacity.

2. Conservation: For each internal vertex $v$ (not $s$ or $t$):

{{ sum }}

Meaning the sum of the flow into each vertex should be equal to the sum of the flow out of that vertex.
